# Security Policy

## Supported Versions

This project currently supports the latest version on the `main` branch.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Instead, report privately to the maintainers with:
- A clear description of the issue
- Reproduction steps or proof of concept
- Impact assessment
- Suggested remediation (if available)

We aim to acknowledge reports within 72 hours and provide a remediation plan as quickly as possible.
